# notebooks/AGENTS.md

Exploratory notebooks for agent development and research.

## Structure

```
notebooks/
├── agents/      # Agent experiments
├── templates/   # Notebook templates
└── *.ipynb      # Notebook files
```

## Conventions

- Keep reproducible — avoid local machine state
- No secrets in cells or outputs
- Short, focused cells
- Clear noisy outputs before committing

## Usage

- VS Code Jupyter or Deno Jupyter kernel
- Base new notebooks on `templates/agent.ipynb`
